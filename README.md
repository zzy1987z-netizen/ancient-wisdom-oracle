# Simple Web Scraper for Quotes

A Python script designed to efficiently scrape famous quotes from the website 'quotes.toscrape.com'. This project serves as a practical demonstration of web crawling and data extraction skills.

## Features

*   **Automatic Pagination:** The script automatically navigates through all pages of the website to ensure complete data collection.
*   **Data Extraction:** For each quote, it accurately extracts three key pieces of information:
    *   The full text of the quote.
    *   The name of the author.
    *   Associated tags.
*   **Structured Output:** All scraped data is saved into a structured file format (e.g., CSV or JSON) for easy analysis and use.
